ksnip Docker container
======================

## Installation

```bash
docker-compose build
```

## Running the container

```bash
docker-compose up
```

- Visit <http://localhost:6080/>
- Run `qtcreator`
- Open project `/code/ksnip/CMakeLists.txt` (*File* menu)
- Press <kbd>Ctrl + B</kbd> to build and run ksnip
